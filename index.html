Welcome!

Dinky is a tiny web server that's able to serve a single file which you provide on the command line.

How tiny? 1808 bytes tiny. That's 8.3 times smaller than the favicon.ico on this page [1]

It's written in Zig, for Linux, and depends on nothing but a few system calls.

Building and running

You can build dinky on a x86_64 Linux machine with: 
make install
To serve index.html on the default port 8088: 
make serve
Which simply runs ./dinky 8088 index.html text/html

Dinky is obviously not intended for production use (no https, single-file serving), as it's just a little elf-golfing experiment.
